# FULivePC

FULivePC 是 Windows 平台上，集成相芯人脸跟踪及视频特效开发包（简 Nama SDK）的集成示例。

集成包括面部跟踪、美颜、Animoji、道具贴纸、AR面具、表情识别、音乐滤镜、背景分割、手势识别、哈哈镜、人像光照以及人像驱动等功能。
